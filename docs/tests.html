Tests¶

Tests specific to the user actions of uploading video content in browser.

class upload_video_tests.UploadVideoTest(methodName='runTest')¶

Represents all the test cases for the page where user uploads video.

test_based_on_current_date_should_find_next_sunday_date()¶

The test confirms that the next Sunday should fall on a correct calendar date.

test_next_sunday_date_should_be_sometime_in_the_future()¶

This should verify that the next sunday date is not in the past.

Based on a given date, the test should confirm that the date to be displayed to the user is equal to some day in the future. The future time may also be the current day – the current Sunday.

test_upload_url_responds_ok()¶

Test to verify that ‘upload’ page returns correct response.

Based on the hard coded uri string, ‘/upload’, the expected response status should be 200 (OK).

test_upload_video_contains_paragraph_with_instructions()¶

This test is verifying that the first paragraph exists.

The content should contain the string ‘submit your video’ somewhere within the body of the page.

Note

This is essentially testing for an existence of a constant. This kind of test is of a dubious value and should probably be deleted.
